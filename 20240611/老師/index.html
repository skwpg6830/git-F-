01.Sass(SCSS) 預前處理器

Sass 目的減少 CSS 的重複性工作，需編譯成原生 CSS 才能使用。(瀏覽器看不懂 Sass(SCSS))

Sass 可以使用變數、迴圈、條件判斷、函式、mixins、嵌套等功能...。

02.變數

變數用 $ 宣告，大括號裡面是區域變數，最外面是全域變數。

變數的底線 _ 和連結線 - 是相同的，$bg_dark、$bg-dark 是一樣的變數。

使用變數可以維持網站的一致性，只需修改變數的值，不用刻意用 ctrl+f 搜尋值修改。

04.嵌套巢狀(nested)

& 可以替換父選擇器。

不要過度使用嵌套，考慮用最少的選擇器，選擇到想要的位置，維持在 3 層內。

過度嵌套會造成編譯後選擇器過長的問題，不易閱讀，且 CSS 容量會較大。

05.extend %

使用 % 來定義名稱，引用時使用 @extend。

繼承可以定義一組相同的樣式，給其他選擇器繼承使用。

繼承被編譯出去會使用群組選擇器，樣式碼會較少。
